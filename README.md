# Seller Portal

## Development Requirement
 * ruby 2.1+
 * php 5.5+
 * composer
 * npm
 * *nix OS


## How to get html
 1. open this directory in terminal
 2. start server by run command `./server.sh`
 3. open web and go to url `http://localhost:5000/index.html` (see html file in `public` directory)
